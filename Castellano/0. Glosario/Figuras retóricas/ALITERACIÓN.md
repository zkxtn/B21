# ALITERACIÓN #card

Consiste en la repetición o reiteración de determinados sonidos dentro del texto escrito, con el fin de obtener una mayor expresividad o impacto sonoro.  
^1681128084643

>Ej.
>“**M**i **m**amá **m**e **m**ima”
>“Bajo **el** a**la** a**le**ve d**el** **le**ve abanico”
