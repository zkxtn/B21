# PARALELISMO  #card 
Es una figura retòrica que consiste en la repetición de una misma estructura gramatical.   

>Ejemplos: 
>- Tu frente serena y firme, tu risa suave y delicada 
>- Errado lleva el camino, errada lleva la guia
